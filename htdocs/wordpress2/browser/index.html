JavaScript

DAILY BLOGでは、より便利にサイトをご利用いただくため、JavaScriptを使用しています。

ウェブブラウザの設定でJavaScriptを無効にしている場合、正しく表示されない場合や機能しない場合があります。　DAILY BLOGをより快適にご利用いただくためには、お使いのウェブブラウザの設定でJavaScriptを有効にしてください。

スタイルシート

DAILY BLOGではスタイルシートを使用しています。ウェブブラウザの設定でスタイルシートを無効にしている場合、DAILY BLOGが正しく表示さません。ウェブブラウザの設定でスタイルシートを有効にしてご利用ください。

Cookieの使用について

DAILY BLOGでは、サービスをより便利にご利用いただくため、Cookieを使用しています。　DAILY BLOGの機能をすべてご利用いただくためには、お使いのブラウザの設定画面にて、Cookieを利用可能な設定にしてください。

通信速度

DAILY BLOGを快適にご利用いただくためには、1.5Mbps以上（ADSL、ケーブルなど）の通信速度でご利用ください。

画面解像度

DAILY BLOGを快適にご利用いただくためには、256以上表示可能な、1024×768ピクセル以上の解像度でご覧ください。
