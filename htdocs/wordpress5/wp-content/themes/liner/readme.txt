== Description ==

Liner is a free WordPress theme for business. This theme is clean, minimal and responsive layout. So this theme optimizes for every device. Liner is supporting customize of the text color, link text color, logo, and a header image. Furthermore, the front page can define the widget areas of the number of any columns.
